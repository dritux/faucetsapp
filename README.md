
#### Versions:
nvm install v10.16.3

#### Dependencies
sudo apt-get install android-tools-adb 


#### Android Studio
```
cd /usr/local/android-studio/bin/ && ./studio.sh 
cd ~/genymotion && ./genymotion
```

#### Running
```
adb devices
adb connect {IP}:{PORT}

react-native run-android
```
